# Darshan-Ease

Darshan-Ease is a simple client-server web application built with **JavaScript, HTML, and CSS**.  
The repository is organized into separate `client` and `server` folders, making it easy to maintain and extend.

## Features

- Separate frontend and backend structure
- Clean project organization
- Easy to extend with new pages, APIs, and functionality
- Suitable for learning and full-stack development practice
