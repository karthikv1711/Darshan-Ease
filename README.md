## 📖 Overview

Darshan-Ease is a full-stack web application designed using a clean client-server architecture. The project follows modern development practices by separating the frontend and backend into independent modules, making the application scalable, maintainable, and easy to extend.

Whether you're learning full-stack development or building production-ready applications, this project provides a solid foundation.

## 🚀 Getting Started

### Prerequisites

Install the following software before running the project.

- Node.js (Latest LTS)
- npm
- Git

---

## 📥 Installation

### Clone Repository

```bash
git clone https://github.com/karthikv1711/Darshan-Ease.git
```

Move into the project directory.

```bash
cd Darshan-Ease
```

---

### Install Backend Dependencies

```bash
cd server
npm install
```

---

### Install Frontend Dependencies

```bash
cd ../client
npm install
```

---

## ▶ Running the Project

### Start Backend

```bash
cd server
npm start
```

or

```bash
npm run dev
```

---

### Start Frontend

```bash
cd client
npm start
```

Open your browser and visit:

```
http://localhost:3000
```

*(Update the port if your project uses a different one.)*
